# 🥋 少林寺拳法 - けんしのこころえゲーム

6歳向けの少林寺拳法学習ゲームです。楽しく遊びながら、拳士の心得を学べます。

## 🎮 ゲームモード

### 📝 クイズモード
4つの答えから正しいものを選ぶクイズ形式です。基本的な知識を楽しく学習できます。

### 🔗 マッチングゲーム
言葉と意味をマッチングさせるゲームです。クリックして正しいペアを見つけましょう。

### ✏️ うめもじモード
（　）に正しい言葉を入力する穴埋め問題です。漢字や読み方を練習できます。

### 🎲 ミックスモード
すべての問題タイプがランダムに出題されます。総合的な学習に最適です。

## 📚 学習内容

### 拳士の心得 6カ条

1. **脚下照顧（きゃっかしょうこ）**
   - 靴を揃えるなど、小さなことを大切にする

2. **合掌礼（がっしょうれい）**
   - 少林寺拳法の挨拶。相手を敬う気持ちが大切

3. **作務（さむ）**
   - 掃除などの日常作業。心もきれいにする

4. **服装（ふくそう）**
   - きれいな服装と身だしなみ

5. **態度（たいど）**
   - 礼儀正しく、明るく元気に学ぶ姿勢

6. **言葉（ことば）**
   - 丁寧な言葉遣い

## 🌟 特徴

- **4種類のゲームモード** - 飽きずに楽しく学習
- **カラフルで楽しいデザイン** - 子供が喜ぶUI
- **アニメーション効果** - 正解・不正解が一目でわかる
- **音声フィードバック** - 正解すると楽しい音が鳴る
- **スマホ対応** - スマートフォンやタブレットでも快適
- **進捗表示** - 今どこまで進んでいるか一目でわかる

## 🚀 使い方

1. ゲームを開く
2. 好きなゲームモードを選ぶ
3. 問題に答える
4. 全問終了後、成績が表示されます

## 🎯 遊び方

### クイズモード
1. 問題を読む
2. 正しいと思う答えをクリック
3. フィードバックを確認
4. 「つぎへ」ボタンで次の問題へ

### マッチングゲーム
1. 左側の「ことば」をクリック
2. 右側の対応する「いみ」をクリック
3. 全部マッチングできたら次へ

### うめもじモード
1. 問題を読む
2. （　）に入る言葉を入力
3. 「こたえをみる」ボタンをクリック
4. フィードバックを確認
